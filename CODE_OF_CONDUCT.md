# Code Of Conduct

This project welcomes practical, technical feedback from INAV users, FPV
builders, and Blackbox-log reviewers.

## Expected

- Be specific and constructive.
- Include aircraft setup, log context, and reproduction notes when possible.
- Respect log privacy and location privacy.
- Treat generated tuning suggestions as review material, not commands to apply
  blindly.

## Not Welcome

- Personal attacks, harassment, or insults.
- Unsafe tuning advice presented as proven.
- Spam, fake engagement, or unrelated promotion.

Moderation decisions are made to keep review focused, safe, and useful.
