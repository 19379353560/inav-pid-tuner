# Roadmap

## Validation Stage

- Collect anonymized INAV Blackbox logs from clean and noisy builds.
- Compare rule output with experienced tuner feedback.
- Track logs and rule notes in https://github.com/19379353560/inav-pid-tuner/issues/1.

## Analysis Stage

- Improve gyro-noise, D-term noise, and PID-error metrics.
- Add clearer before/after comparison output.
- Document when a recommendation should be ignored.

## Product Stage

- Make the browser UI easier to review safely.
- Add example logs or synthetic fixtures where licensing/privacy allows.
- Publish a simple validation report for each rule update.
