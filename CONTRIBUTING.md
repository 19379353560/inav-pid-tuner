# Contributing

Thanks for helping validate INAV PID Tuner. The project is most useful when its
recommendations are checked against real Blackbox logs from different aircraft.

## Helpful Contributions

- Anonymized `.bbl` or `.bfl` logs from clean and noisy builds.
- Before/after logs after PID or filter changes.
- Notes about frame size, props, motors, firmware version, and known vibration
  sources.
- Safer tuning-rule suggestions and UI improvements.

## Privacy

Please remove private GPS/home-position data before sharing logs publicly. If
you are unsure, open an issue with screenshots or summary metrics first.
