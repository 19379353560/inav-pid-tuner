# Safety

INAV PID Tuner provides analysis and cautious suggestions. It is not an
automatic tuning authority.

## Before Applying Suggestions

- Review every CLI line before applying it.
- Make small changes and test one change set at a time.
- Watch motor temperature and aircraft behavior after filter or PID changes.
- Keep a backup of the previous tune.
- Do not use recommendations from logs with corrupted data, unsafe flights, or
  unknown setup details.

## Log Privacy

- Remove private GPS/home-position data before sharing public logs.
- Share screenshots or summary metrics when full logs cannot be public.
- Do not publish another pilot's logs without permission.
