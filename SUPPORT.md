# Support and Review Channels

PID/filter tuning suggestions.

## Useful Support Topics

- Anonymized INAV Blackbox logs from different airframes.
- Better validation rules for fixed wing, freestyle, and long-range builds.
- Sample metrics for demo mode and regression checks.
- UI wording that helps pilots review recommendations safely.

For log-sharing help, include firmware version, aircraft class, filter settings,
flight condition, and whether the log may be used as public sample data.
